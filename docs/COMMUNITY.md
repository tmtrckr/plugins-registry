# Community

## GitHub Discussions

Maintainers can enable **GitHub Discussions** for this repository:

1. Go to **Settings** → **General** → **Features**
2. Enable **Discussions**
3. Create categories, for example:
   - **Announcements** – Registry and policy updates
   - **Q&A** – Questions about adding or using plugins
   - **Ideas** – Feature requests and ideas for the registry
   - **Show and Tell** – Share your plugins or integrations

After enabling, the **Discussions** tab will appear in the repository. Link to it from the README (e.g. `https://github.com/tmtrckr/plugins-registry/discussions`).

## Communication Channels

- **Issues**: Bug reports and feature requests (use the issue templates).
- **Pull requests**: Plugin submissions and documentation changes.
- **Discussions** (when enabled): General questions and community chat.

## Contributor Recognition

Contributors are listed in the repository's **Contributors** tab. Maintainers may optionally configure the [All Contributors](https://allcontributors.org/) bot or list top contributors in the README.
